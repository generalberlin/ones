# Ones Filo

Ones Filo için hazırlanmış, hafif yapıda ve statik yayınlamaya uygun kurumsal araç kiralama web sitesi.

## İçerik
- Kurumsal ana sayfa yapısı
- Mobil uyumlu tasarım
- Filtrelenebilir araç listesi
- Telefon ve WhatsApp hızlı iletişim butonları
- Yerel varlıklarla çalışan sade dosya yapısı

## Kullanım
1. Dosyaları hosting alanına veya bir Git deposuna yükleyin.
2. `index.html` dosyasını başlangıç sayfası olarak yayınlayın.
3. İletişim numarası ve içerik metinlerini ihtiyaca göre güncelleyin.

## Dosyalar
- `index.html` : Ana sayfa yapısı
- `styles.css` : Tasarım ve responsive stiller
- `script.js` : Filtreleme ve listeleme işlemleri
- `cars.js` : Araç verileri
- `assets/` : Logo, video ve araç görselleri
